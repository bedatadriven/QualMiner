Technical aspects of the data

This part might seem to be too technical for readers who are mainly interested in the results. The reason why this page exists is that the information here can be essential for the analysts working with the data.

The inspectdf package is used to generate the visualizations in this page.

Figure 3: The column types in the data frame

Almost all columns are character vector, except required.

Figure 4: Summarising categorical features in the data frame

We see that the response field has many unique levels which has a solid color in the chart.

Figure 5: See the most frequently occurring categorical level in each column.

Figure 6: The prevalance of NAs (missing values) in the colums.

Figure 7: The size of each column in the data frame.

As seen in the plot, the response column has the most size because it holds long text fields.
